Project Overview

Traditional loan approval processes in many financial institutions suffer from significant gaps including lack of transparency, inconsistent decision-making, and inability to adapt to nuanced applicant profiles. This project addresses these challenges by developing an intelligent, automated loan application and approval system.

The system leverages machine learning to provide transparent, consistent, and explainable loan decisions while offering comprehensive insights into approval or denial reasoning. Our solution transforms the traditional lending process into a data-driven, efficient, and fair system.

🔧 Technical Implementation

Our model employs a Random Forest Classifier with 100 estimators, chosen for its excellent performance with structured data and mixed feature types. The model provides built-in feature importance metrics, crucial for explaining decisions to stakeholders.

Model Training & Evaluation Python

Our model's feature importance analysis reveals that CIBIL score dominates decision-making (80%+ influence), followed by loan term, amount, income, and asset values. This aligns perfectly with real-world lending practices, where creditworthiness and repayment capacity are paramount.

Automated Processing Pipeline

Our sophisticated backend pipeline seamlessly integrates data collection, processing, and prediction generation. The system automatically loads new loan applications, processes them through our trained model, and generates comprehensive results with approval probabilities.

Complete Prediction Pipeline Python 
import pandas as pd
import joblib
from google.colab import drive, auth
import gspread
from google.auth import default

def predict_loan_approval():
    # Mount Google Drive and load trained model
    drive.mount('/content/drive')
    model = joblib.load('/content/drive/MyDrive/loan_model.pkl')
    
    # Load application data from Google Sheets
    sheet_id = '1VGRF8daNSc3RiED0GqIzNjLjt1XLYHT7lux5x83s0Dc'
    sheet_url = f"https://docs.google.com/spreadsheets/d/{sheet_id}/gviz/tq?tqx=out:csv"
    new_data = pd.read_csv(sheet_url)
    
    # Data preprocessing and feature mapping
    column_mapping = {
        'No of dependents': 'no_of_dependents',
        'Education': 'education',
        'Employment': 'self_employed',
        'Income': 'income_annum',
        'Loan Amount': 'loan_amount',
        'Loan Term': 'loan_term',
        'Credit Score': 'cibil_score'
        # ... additional mappings
    }
    
    processed_data = new_data.rename(columns=column_mapping)
    
    # Intelligent data type conversion
    processed_data['education'] = processed_data['education'].map({
        'Graduate': 1, 'Not Graduate': 0
    }).fillna(0)
    
    processed_data['self_employed'] = processed_data['self_employed'].map({
        'Yes': 1, 'No': 0
    }).fillna(0)
    
    # Generate predictions with probabilities
    predictions = model.predict(processed_data)
    prediction_probs = model.predict_proba(processed_data)[:, 1]
    
    # Enhance results with detailed insights
    new_data['Predicted_Approval'] = ['Approved' if x == 1 else 'Rejected' for x in predictions]
    new_data['Approval_Probability'] = prediction_probs
    new_data['Approval_Probability_Pct'] = (prediction_probs * 100).round(2)
    
    # Save comprehensive results
    output_path = '/content/drive/MyDrive/LOANAPP_Predictions.csv'
    new_data.to_csv(output_path, index=False)
    
    return True

# Execute the pipeline
predict_loan_approval()
